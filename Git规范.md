# Git规范

## 提交规范

|     类型      | 描述                                                  |
| :-----------: | ----------------------------------------------------- |
| build/release | 编译相关的修改,例如发布版本                           |
|     chore     | 其他修改,日常事务的修改                               |
|      ci       | 持续集成修改                                          |
|     docs      | 文档修改                                              |
|     feat      | 新特性、新功能,比如改变构建流程、或增加依赖库、工具等 |
|      fix      | 修改BUG                                               |
|     perf      | 优化相关:提升性能、体验                               |
|   refactor    | 代码重构                                              |
|    revert     | 回滚到上一个版本                                      |
|     style     | 代码格式修改,注意不是css修改                          |
|     test      | 测试用例修改                                          |
